# Claude AI 课程学员社区

欢迎！这是「从零上手 Claude」付费课程学员的共享资源库。

**加入方式：** 通过课程微信大群获取入群方式。

---

## 4 个兴趣方向

| 方向 | 定位 |
|------|------|
| [财富管理](./财富管理/) | 北美华人理财/税务/保险/退休规划，Claude Code 知识库 |
| [金融分析](./金融分析/) | 股票/市场/基金分析自动化，prompt pipeline 模板 |
| [内容创作](./内容创作/) | 公众号/视频脚本/PPT 生成，Claude Cowork 工具箱 |
| [其他](./其他/) | 职场自动化及各类 Claude 应用（邮件/报告/日常工具） |

每个方向下有三个目录：
- `skills/` — 可直接装进 Claude Code 的 skill 文件
- `prompts/` — prompt 模板，复制后直接用
- `wiki/` — 知识库条目（Markdown）
- `笔记/` — 会议记录、idea、心得体会

## 怎么用

**不会 git？**
点右上角绿色 `Code` 按钮 → `Download ZIP`，解压后找对应目录的文件即可。

**会 git？**
欢迎直接提 PR 贡献内容。

---

## 运营方式

- **微信大群**：所有付费学员（S1 + S2）在同一个群，活动通知都在这里
- **兴趣 Zoom**：每个方向每月 1-2 次小会，感兴趣的报名参加，自由讨论 + 自愿演示
- **GitHub 存档**：Zoom 里分享的好东西，组长整理后上传到这里，大家下载用

## 贡献内容

不需要懂 GitHub，直接在微信群发给组长，组长帮你整理上传。
