# 组长使用指南：如何向社区库添加内容

> 全程用 Claude Code 操作，不需要手敲 git 命令。

---

## 前提准备（只需做一次）

### 1. 安装 Claude Code

去 [claude.ai/code](https://claude.ai/code) 下载安装，登录你的 Claude 账号。

### 2. 克隆社区库

打开 Claude Code，直接说：

```
帮我克隆 https://github.com/claude-ai-community/community.git 到本地
```

Claude 会帮你克隆并进入目录。

> 如果之前已经克隆过，说：`帮我同步最新的 community 仓库内容`

### 3. 安装 skill（只需做一次）

在 community 目录下的 Claude Code 中说：

```
/install .claude/skills/内容上传.md
```

---

## 日常使用：上传内容

### 第一步：打开 Claude Code，进入 community 目录

对 Claude 说：`进入 community 目录`

### 第二步：上传内容

说 `/内容上传`，然后告诉 Claude 你要上传什么。

**示例 1：粘贴文字内容**
```
/内容上传 内容创作

帮我把下面这个 prompt 加进来：

你是一个专业的小红书文案写手...（粘贴内容）
```

**示例 2：本地文件**
```
/内容上传 财富管理 文件：桌面上的税务指南.md
```

**示例 3：GitHub 项目**
```
/内容上传 https://github.com/某人/某项目
```

**示例 4：不确定分类，让 Claude 自动判断**
```
/内容上传

（粘贴内容）
```

Claude 会告诉你它判断放在哪里，确认后写入。

### 第三步：提交到 GitHub

文件写入后，对 Claude 说：

```
帮我提交并推送到 GitHub
```

Claude 会自动完成 `git add` → `git commit` → `git push`，你只需确认一下。

## 目录说明

| 目录 | 放什么 |
|------|--------|
| `skills/` | Claude Code skill 文件，别人可以直接 `/install` 使用 |
| `prompts/` | Prompt 模板，复制到 Claude 对话框直接用 |
| `wiki/` | 知识库文章、长篇参考资料 |
| `笔记/` | Zoom 会议记录、idea、心得分享 |

四个小组：`财富管理` / `金融分析` / `内容创作` / `其他`

## 常见问题

**Q：我对分类不确定怎么办？**
A：不用管，Claude 自动判断，写入前会告诉你，你确认就行。

**Q：push 时提示权限不足？**
A：联系 Austin 开通 GitHub 写入权限。

**Q：内容是截图或聊天记录怎么办？**
A：描述给 Claude，或者把文字复制给它，它会帮你整理好再写入。

**Q：没有 GitHub 账号怎么办？**
A：把内容发给 Austin，他来上传。
