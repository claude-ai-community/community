# 组长使用指南：如何向社区库添加内容

## 前提准备

### 1. 安装 Claude Code

去 [claude.ai/code](https://claude.ai/code) 下载安装，登录你的 Claude 账号。

### 2. 克隆（下载）社区库

```bash
git clone https://github.com/austinxyz/community.git
cd community
```

> 如果之前已经克隆过，先同步最新内容：
> ```bash
> git pull
> ```

### 3. 安装 skill

在 `community` 目录下打开 Claude Code，运行：

```
/install .claude/skills/内容上传.md
```

---

## 日常使用流程

### 第一步：在 Claude Code 中打开 community 目录

```bash
cd community
claude
```

### 第二步：调用 skill 上传内容

在 Claude Code 中输入 `/内容上传`，然后告诉 Claude 你要上传什么。

**示例 1：粘贴文字**
```
/内容上传

帮我把下面这个 prompt 加到对应目录：

你是一个专业的小红书文案写手，擅长...（粘贴内容）
```

**示例 2：本地文件**
```
/内容上传 文件路径：C:\Users\你的名字\Desktop\税务指南.md
```

**示例 3：GitHub 项目**
```
/内容上传 https://github.com/某人/某项目
```

**示例 4：指定小组**
```
/内容上传 财富管理

（内容）
```

或者更精确：
```
/内容上传 小组：金融分析 类型：wiki

（内容）
```

Claude 会自动判断放到哪个目录，并在写入前让你确认。

### 第三步：提交并推送到 GitHub

Claude 写入文件后会给你提示命令，直接复制运行：

```bash
git add 财富管理/wiki/美股税务申报指南.md
git commit -m "docs: 添加美股税务申报指南"
git push
```

## 目录说明

| 目录 | 放什么 |
|------|--------|
| `skills/` | Claude Code skill 文件，其他人可以直接 `/install` 使用 |
| `prompts/` | Prompt 模板，复制到 Claude 对话框直接用 |
| `wiki/` | 知识库文章，长篇参考资料 |
| `笔记/` | Zoom 会议记录、idea、讨论心得 |

四个兴趣方向：`财富管理` / `金融分析` / `内容创作` / `其他`

## 常见问题

**Q：我没有 GitHub 账号，可以用吗？**
A：需要 GitHub 账号才能 push。如果没有，把内容发给 Austin，由他上传。

**Q：我对分类不确定怎么办？**
A：Claude 会自动判断并告诉你，你确认就行。也可以在调用时直接说"放到财富管理/wiki"。

**Q：push 时提示权限不足？**
A：联系 Austin 开通 GitHub 仓库的写入权限。

**Q：内容是聊天记录截图怎么办？**
A：描述截图内容给 Claude，它会帮你整理成规范格式再写入。
