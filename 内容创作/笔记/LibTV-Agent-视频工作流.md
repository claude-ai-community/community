# LibTV Agent：100个AI视频工作流重组为Skill

## 工具简介

LibTV 推出 Agent 功能并内置 Skill Hub，提供 100+ 视频 Skill，覆盖武侠电影、皮克斯动画广告、电商口播等类型。

## 核心功能

- 输入想法后，Agent 分析需求并询问方向
- 自动生成视频分镜并串联成完整节点工作流
- 每个节点可查看和修改提示语
- 生成后自动启动自查机制，检测并返修有问题的镜头

## 适用场景

武侠电影、皮克斯风动画广告、电商口播视频等
